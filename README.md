Black Space это сообщество по игре Space Station 14 которая является ремейком SS13, которая работает на собственном движке [Robust Toolbox](https://github.com/space-wizards/RobustToolbox), написанном на C#.

Это репозиторий сервера по тематике Союза Советских Социалистических Планет, который показывает каким был бы СССР в будущем.

## Документация

На официальном сайте с [документацией](https://docs.spacestation14.io/) имеется вся необходимая информация о контенте SS14, движке, дизайне игры и многом другом. Также имеется много информации для начинающих разработчиков.

## Контрибьют

В случае если вы хотите добавить новый контент будет лучше, если сначала вы предложите его в [основной репозиторий](https://github.com/space-wizards/space-station-14) или обсудите его необходимость на нашем сервере [Discord](https://discord.gg/FeurRMmVDd).

## Сборка

1. Склонируйте этот репозиторий локально
2. Запустите `RUN_THIS.py` для инициализации подмодулей и скачивания движка.
3. Скомпилируйте проект.

[Более подробная инструкция по запуску проекта.](https://docs.spacestation14.io/getting-started/dev-setup)
